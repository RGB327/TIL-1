# 리눅스
---
## 리눅스란?
> UNIX 기반의 오픈 소스 운영 체제로 컴퓨터 하드웨어와 소프트웨어 리소스 관리, 사용자와 프로그램 간의 인터페이스를 제공함.

### 리눅스의 구조
1. **커널**
    - 프로세스, 메모리, 파일, 장치 등 컴퓨터의 **모든 자원을 관리**하는 **운영체제의 핵심**으로 **정체성**을 **결정**지음
2. **셀**
    - 일종의 **명령어 해석기**로 사용자와 장치 사이에서 명령어를 번역하고 **커널에게 전달**함
3. **사용자 프로그램**
    - 프로그래밍 개발 도구, 문서 편집 도구 등 일반적으로 사용하는 **응용프로그램**

### 우리는 왜 리눅스를 사용하는가
1. **구조 안정성**
    - 리눅스는 매우 **안정적**이며 **충돌이 잘 발생하지 않음**
    - **오픈소스**로 되어있는 리눅스 특성상 **많은 사람들이 버그를 찾고 수정**하기 때문
2. **보안적 요인**
    - 시스템 파일 권한과 소유권을 세밀하게 제어할 수 있어 바이러스 및 해킹 위협에 상대적으로 안전
3. **시스템이 가볍다**
    - (개발에)불필요한 기능이 적어 비교적 **가벼움**
4. **무료로 사용 가능하다**
    - 오픈소스에 **무료**라 동작이 자유롭고 **비용측에서도 이득임**
5. **서버와 동일한 환경** 
    - 대부분의 서버에서는 리눅스를 사용하기 때문에 **배포**와 **개발**이 **자연스럽게 이어지게** 하기 위해 사용

## 리눅스의 특징과 종류

- **오픈소스**
- **안정성 및 보안성**
- **유연성**
- **호환성**
...


위에 설명한 리눅스를 사용하는 이유 중 대부분이 리눅스의 특징이다.

### 리눅스의 종류

**Ubuntu**
- 설치가 쉽고 자료가 많음
- 개발용으로 많이 사용(GPT피셜)

**Rocky Linux**
- 안정성 중심
- 서버에서 주로 사용

**Kali Linux**
- 보안/해킹 테스트용 리눅스

## 리눅스의 대표 명령어
- **ls** - 현재 디렉터리의 파일을 볼 수 있음
- **cd (경로)** - 디렉터리 이동(..은 상위 디렉터리로 이동)
- **pwd** - 현재 위치의 절대 경로 표시
- **mkdir (디렉터리 이름)** - 디렉터리를 만듬
- **rm (파일명)** - 파일 삭제(디렉터리는 -r 옵션 필요)
- **rmdir** - 디렉터리 삭제(파일이 있는 디렉터리는 -r)
- **cat (파일명)** - 파일 내용 확인
- **touch (파일명)** - 파일 생성 및 수정
- **mv** - 파일이나 디렉터리의 이름을 변경하거나 다른 디렉터리로 옮길 때 사용
- **clear** - 터미널 화면 청소
