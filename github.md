# 깃허브 사용법
-----
## 깃의 명령어들

### 초기 설정 명령어
- **git init**
    - 현재 폴더를 Git 저장소로 초기화함
- **git clone <깃허브 링크>**
    - GitHub 저장소를 복제해서 로컬에 가져옴
---
### 저장 및 기록 명령어
- **git add <파일명>**
    - 변경된 파일을 스테이징 영역으로 올림 (. 압력 시 변경된 모든 파일 전체 선택)
- **git commit -m**
    - add를 통해 스테이징 영역으로 올린 코드들을 로컬 저장소에 메세지와 함께 확정(-m 안넣으면 로딩만 돌다 오류남)
- **git status**
    - 현재 변경 상태 확인(수정/추가/커밋 여부)
- **git log**
    - 지금까지의 커밋 기록 확인
---
### 원격 저장소(Github) 연결 명령어
- **git push**
    - 로컬 커밋을 깃허브에 올림
- **git pull**
    - 깃허브의 변경 내용을 가져와 병합함
- **git fetch**
    - 깃허브에서 변경 내용을 가져오기만 함(병합x)
